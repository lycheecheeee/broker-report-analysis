核心模塊說明

  • 後端服務 (backend.py): Flask API + PDF解析 + AI集成
  • 前端界面 (universal_pdf_dashboard.html): Glassmorphism UI + 實時進度條
  • 數據庫 (broker_analysis.db): SQLite 存儲分析結果
  • AI集成: OpenRouter Qwen 2.5 - 智能字段提取 + 繁體中文強制輸出

快速開始

  1. 安裝依賴: pip install flask flask-cors PyPDF2 requests
  2. 啟動服務: python backend.py
  3. 訪問: 當前網址
  4. 測試帳號: lychee / lycheechee2026
